Database


    Avialable Courses:

  • Introduction to Databases and Programming This course focuses on the uses of relational and object-oriented databases for storing and managing information. Topics include computer database terminology and the evolution of the modern database.

  • Database Management Fundamentals and Administration In this course you will you will gain a basic understanding of database models and environments as they learn to manage database components. Topics discussed include data input, data sorting (tables, filters and queries), database troubleshooting and database security.

  • Database System Programming This course emphasizes the computer programming aspects of database management. Through instruction in advanced SQL and other programming languages, students learn to design database system functions, including web applications, database triggers, report forms and security configuration.
